Cliver Excel 5.*

Overview

CliverExcel is an .NET API for creating and editing excel files.
It is based on the NPOI lib and allows accessing the NPOI's API, besides its own. Its goal is to provide an API more advanced than NPOI's.

Compatibility
CliverExcel is compatible with .NET Standard 2.0.

API

All the methods are plainly accessible through the class Cliver.Excel. In this class their scope is defaulted to the opened workbook and currently picked worksheet.

Alternatively, all the basic methods are presented as extension methods of the NPOI types IWorkbook, ISheet, IRow and ICell.
(!)The extension methods start with '_' to be easily distinguished from the NPOI natives.

(!)In CliverExcel, cell, row and sheet indexes are always 1-based, while in the NPOI API they are mostly 0-based. As both API's coexist, be careful of the arguments and output of the methods you are using. In the CliverExcel methods' arguments, column and row indexes are always represented by 'x' and 'y' respectively.

Excel.Column

The class Excel.Column exposes methods on columns analogous to the extension methods of NPOI's IRow. It is native to CliverExcel as has no underlying NPOI type.
