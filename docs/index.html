Cliver Excel 4.*

Overview

CliverExcel is an API for creating and editing excel files.

It wraps the NPOI lib and, besides its own API, allows accessing the NPOI methods, too.

CliverExcel's goal is to provide an API more handy and advanced than NPOI's.

Compatibility
CliverExcel is compatible with .NET Standard 2.0.

API

All the methods are plainly accessible through the class Cliver.Excel. In this class their scope is restricted by the current workbook and worksheet.

Alternatively, they are also presented as extension methods in the NPOI types IWorkbook, ISheet, IRow and ICell. (!)The extension methods start with '_' to be easily distinguished from the NPOI natives.

(!)In CliverExcel, cell, row and sheet indexes are always 1-based, while in the NPOI API they are mostly 0-based. As both API's coexist, be careful of the arguments and output of the method you are using. In the CliverExcel methods, column and row indexes are always represented by x and y respectively.

Column

The class Column exposes methods on columns analogous to the extension methods of NPOI's IRow. It is purely native to CliverExcel as has no underlying NPOI type.
